# CampusFlow Services

Each service is a TypeScript Node.js package with its own source folder, package manifest, and deployment boundary.

## Service Ports

| Service | Port |
| --- | --- |
| API Gateway | `4000` |
| Auth Service | `4001` |
| User Service | `4002` |
| Attendance Service | `4003` |
| Fee Service | `4004` |
| Event Service | `4005` |
| Notification Service | `4006` |
| Academic Service | `4007` |
| Library Service | `4008` |
| Document Service | `4009` |

## Implementation Order

1. Auth Service
2. API Gateway
3. User Service
4. Attendance Service
5. Fee Service
